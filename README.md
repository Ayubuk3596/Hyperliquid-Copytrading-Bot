## 📊 How Copy Trading Works

The bot watches the source account and checks for new trades. When it sees one, it sends a matching order from your account based on your rules.

Example:

- Trader opens a long position
- Bot sees the trade
- Bot places a similar long position
- Bot uses your size settings
- Bot logs the action

This lets you follow a trader without watching the screen all day.
